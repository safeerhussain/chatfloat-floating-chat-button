A simple and lightweight plugin to add a floating WhatsApp button on your website. Fully customizable via admin settings.

== Description ==
A lightweight and easy-to-use WordPress plugin that adds a floating WhatsApp button on every page of your site. Let visitors instantly reach out via WhatsApp. Ideal for businesses, freelancers, or support teams who want to stay connected with their users in real time.

Easily configure your WhatsApp number and button text from the plugin’s settings page. No coding required.

== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. Navigate to **Settings > WhatsApp Float** to configure your number and button text.
4. Save settings and you’re done!

== Frequently Asked Questions ==
= Does this plugin work with Gutenberg? =
Yes, it works independently of the content editor, so it’s fully compatible with both Gutenberg and Classic Editor.

= Can I customize the appearance? =
You can override the CSS by adding custom styles to your theme or child theme’s CSS.

= Does it support multiple WhatsApp numbers? =
Currently, the plugin supports a single number. Future versions may offer multiple profiles.
